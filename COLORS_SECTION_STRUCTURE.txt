FIELD DEPENDENCIES & LOGIC
═══════════════════════════════════════════════════════════════════════════

1. color_units → Controls visibility of color fields
   • If color_units = 1: Show only color (Color 1)
   • If color_units = 2: Show color, color_2
   • If color_units = 3: Show color, color_2, color_3
   • ... and so on up to 10

2. Artwork Color Selection → Depends on color_units
   • body_artwork_color options: 1-10 (but only 1 to color_units are valid)
   • koti_artwork_color options: 1-10
   • bottom_artwork_color options: 1-10
   • dupatta_artwork_color options: 1-10

3. Print Format Integration
   • Color swatches displayed based on color_units
   • Artwork colors shown in print format
   • Color descriptions printed
